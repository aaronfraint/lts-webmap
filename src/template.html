LTS Analyzer (prototype!)


Legend

  • Dashed lines for high-stress (Bike LTS 3 or 4) roadways
  • Colorful lines for low-stress (Bike LTS 1 or 2) islands of connectivity

Usage

  • Zoom in to an area of interest
  • Click the rectangle icon at the top-right corner of the map to enable drawing
  • Click on the map to draw a polygon around stretch of roadway
  • Double-click your last point to finish the drawing
  • The map will then perform the following actions:
    1. Select the LTS 3 and 4 segments within your polygon
    2. Query the API with the selected IDs to find low-stress islands that intersect the selected segments
    3. Refresh the map display, showing the selected segments in cyan and the merged islands with a yellow underline

Front-end: GitHub Pages

  • MapboxGL.js is used to render the map
  • turf.js is used to perform client-side geoprocessing

Back-end: Digital Ocean

  • Vector tile data is hosted using TileServer GL
  • An API was written in Python with FastAPI
  • Data is stored in a PostgreSQL database
